# Awesome Python Blogs

[toc]

本文将列出一些优质的 Python 博客链接，作为 Python 翻译组的选题来源。这里，将主要包括程序员个人博客和公司博客两种类型。

具体格式为：

- 程序员姓名或公司名称
- 程序员或公司简介，说明与 Python 的关系
- 博客地址

欢迎大家提 PR 推荐，添加更多 Python 优质博客链接。
